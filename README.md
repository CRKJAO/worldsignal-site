# Worldsignal

一个无需数据库、无需 API 密钥的全球新闻热点雷达。系统每小时从国际媒体和区域新闻网络抓取公开 RSS，将报道按时效性、影响力关键词、跨来源印证和地域多样性排序，然后发布为适合 GitHub Pages 的静态网站。

## 本地运行

需要 Node.js 20 或更高版本。

```bash
npm run update
npm run check
npm run serve
```

访问 `http://localhost:4173`。

## 自动更新与部署

1. 将仓库默认分支设为 `main`。
2. 在仓库的 **Settings → Pages → Build and deployment** 中选择 **GitHub Actions**。
3. 手动运行一次 **Update global news** 工作流，之后系统会每小时自动更新。

自动化只保存标题、简短摘要、图片链接和原文链接；读者始终会被导向原出版方。若抓取源临时故障且新数据少于健康阈值，脚本会保留上一版数据，避免网站变空。

## 排序逻辑

- 时效：72 小时内的报道，越新权重越高
- 影响：战争、选举、灾害、重大协议、公共卫生等事件信号
- 印证：不同媒体报道相似事件时提升权重
- 多样性：限制单一地区占据全部版面
- 去重：合并高度相似的标题

新闻排序是机器辅助判断，不代表事实核查或编辑背书。重大事件请交叉阅读原始来源。
